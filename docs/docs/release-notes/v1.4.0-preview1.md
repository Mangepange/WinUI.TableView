# v1.4.0-preview1

> **Preview Release** · November 28, 2025 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.4.0-preview1) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.4.0-preview1)

## 🌟 New Features

- Added support for non-scrollable row headers by [@w-ahmad](https://github.com/w-ahmad) in [#210](https://github.com/w-ahmad/WinUI.TableView/pull/210)
- Added Frozen Columns support by [@w-ahmad](https://github.com/w-ahmad) in [#223](https://github.com/w-ahmad/WinUI.TableView/pull/223)
- Added Row Details support by [@w-ahmad](https://github.com/w-ahmad) in [#228](https://github.com/w-ahmad/WinUI.TableView/pull/228)
- Added Conditional cell styling for better customization by [@w-ahmad](https://github.com/w-ahmad) in [#253](https://github.com/w-ahmad/WinUI.TableView/pull/253)
- Added cell editing events (`BeginningEdit`, `PreparingCellForEdit`, `CellEditEnding`, `CellEditEnded`) by [@w-ahmad](https://github.com/w-ahmad) in [#235](https://github.com/w-ahmad/WinUI.TableView/pull/235)
- Added column reordering functionality (drag-and-drop) by [@w-ahmad](https://github.com/w-ahmad) in [#240](https://github.com/w-ahmad/WinUI.TableView/pull/240)
- Added `ClipboardContentBinding` to customize copy-to-clipboard content by [@w-ahmad](https://github.com/w-ahmad) in [#243](https://github.com/w-ahmad/WinUI.TableView/pull/243)
- Added unified `FontSize` and `FontFamily` across the `TableView` by [@w-ahmad](https://github.com/w-ahmad) in [#254](https://github.com/w-ahmad/WinUI.TableView/pull/254)
- Added standard [Fluent Compact sizing](https://learn.microsoft.com/en-us/windows/apps/design/style/spacing#fluent-compact-sizing) to the `TableView` by [@w-ahmad](https://github.com/w-ahmad) in [#255](https://github.com/w-ahmad/WinUI.TableView/pull/255)
- Added XAML Hot Reload support when adding columns by [@w-ahmad](https://github.com/w-ahmad) in [#203](https://github.com/w-ahmad/WinUI.TableView/pull/203)
- Added filter flyout trigger with right-click by [@SISTF](https://github.com/SISTF) in [#193](https://github.com/w-ahmad/WinUI.TableView/pull/193)
- Implemented new Uno row style to match WinUI by [@w-ahmad](https://github.com/w-ahmad) in [#242](https://github.com/w-ahmad/WinUI.TableView/pull/242)

## What's New

- Use compiled expressions to fetch cell content for sort, filter, and clipboard operations by [@DanielTrommel](https://github.com/DanielTrommel) in [#178](https://github.com/w-ahmad/WinUI.TableView/pull/178)
- Added pt-BR translations by [@josuelopes512](https://github.com/josuelopes512) in [#230](https://github.com/w-ahmad/WinUI.TableView/pull/230)
- Upgraded `Uno.WinUI` to 6.2.87 by [@w-ahmad](https://github.com/w-ahmad) in [#242](https://github.com/w-ahmad/WinUI.TableView/pull/242)
- Exposed `RefreshView` and `RefreshSorting` methods by [@RolandBuchholz](https://github.com/RolandBuchholz) in [#185](https://github.com/w-ahmad/WinUI.TableView/pull/185)
- Exposed `AllowLiveShaping` property to turn off live shaping of the view by [@w-ahmad](https://github.com/w-ahmad) in [#248](https://github.com/w-ahmad/WinUI.TableView/pull/248)

## Native AOT

- Added `IsAOTCompatible` flag to enable Native AOT compatibility by [@licon4812](https://github.com/licon4812) in [#217](https://github.com/w-ahmad/WinUI.TableView/pull/217)

## Fixes

- Suppress `ScrollRowIntoView` during multi-selection by [@SISTF](https://github.com/SISTF) in [#187](https://github.com/w-ahmad/WinUI.TableView/pull/187)

## Breaking Changes

- `CellsPresenter` was renamed to `TableViewRowPresenter` as part of adding Row Details. `TableViewCellsPresenter` became `TableViewRowPresenter`. [#228](https://github.com/w-ahmad/WinUI.TableView/pull/228)

## Behavior Changes

- `TableViewBoundColumn.Binding` now defaults `UpdateSourceTrigger` to `Explicit` (from `Default`) to prevent unintended updates and allow Esc to cancel edits. [#235](https://github.com/w-ahmad/WinUI.TableView/pull/235)
