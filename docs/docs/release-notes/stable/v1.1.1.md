# v1.1.1

> **Stable Release** · May 19, 2024 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.1.1) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.1.1)

## What's New

- Enhanced filter flyout functionality and user interface
- Changed the default `HeaderRowWidth` from 48 to 32 (it was previously 32 until version 1.1.0)
- Made the vertical scrollbar display under the header row
- Implemented auto-resizing column by double-clicking on the resize area

## Fixes

- Fixed column header resize issue when `CanFilter` is set to `false`
- Fixed an issue where column resize was using the next column's min and max widths
