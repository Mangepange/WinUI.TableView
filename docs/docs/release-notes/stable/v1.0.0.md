# v1.0.0

> **Stable Release** · April 20, 2024 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.0.0) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.0.0)

## Key Features

- Auto Generate Columns
- Excel-like column filter
- Export data to CSV or implement your own export logic
- Support for column sorting
- Inline data editing

## Enhancements

- Implemented `AutoGenerateColumns` property change handlers
- Added events for export and copy data to implement custom logic:
  - `ExportAllRowsContent`
  - `ExportSelectedRowsContent`
  - `CopyToClipboard`
