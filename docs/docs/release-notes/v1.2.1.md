# v1.2.1

- **Improved cell & row navigation performance**

- Fixed `CheckBox` & `ToggleSwitch` column `IsReadOnly` [#36](https://github.com/w-ahmad/WinUI.TableView/issues/36), [#44](https://github.com/w-ahmad/WinUI.TableView/issues/44)
- Fixed possible null exception by null annotations [#43](https://github.com/w-ahmad/WinUI.TableView/issues/43), [#47](https://github.com/w-ahmad/WinUI.TableView/issues/47)
- Fixed cell & row navigation behaviors [#45](https://github.com/w-ahmad/WinUI.TableView/issues/45)
- Fixed current cell highlight behavior [#45](https://github.com/w-ahmad/WinUI.TableView/issues/45), [#46](https://github.com/w-ahmad/WinUI.TableView/issues/46)
- Added source link symbol package
