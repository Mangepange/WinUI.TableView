# v1.3.2

> **Stable Release** · July 30, 2025 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.3.2) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.3.2)

## What's New

- Added `TableView.HeaderRowMinHeight` and `TableView.HeaderRowMaxHeight` properties
- Added zh-CN strings by [@lostluna1](https://github.com/lostluna1) in [#161](https://github.com/w-ahmad/WinUI.TableView/pull/161)
- Added zh-TW strings by [@lostluna1](https://github.com/lostluna1) in [#161](https://github.com/w-ahmad/WinUI.TableView/pull/161)
- Fixed ru-RU strings by [@slick1983](https://github.com/slick1983) in [#174](https://github.com/w-ahmad/WinUI.TableView/pull/174)

## Fixes

- Fixed blank values handling in column filter by [@w-ahmad](https://github.com/w-ahmad) in [#162](https://github.com/w-ahmad/WinUI.TableView/pull/162)
- Fixed sort and filter issues with array bindings by [@DanielTrommel](https://github.com/DanielTrommel) in [#166](https://github.com/w-ahmad/WinUI.TableView/pull/166) & [#177](https://github.com/w-ahmad/WinUI.TableView/pull/177)
- Added defer refresh on sorting to keep horizontal scroll offset by [@w-ahmad](https://github.com/w-ahmad) in [#167](https://github.com/w-ahmad/WinUI.TableView/pull/167)
- Fixed cell and row measure issues by [@w-ahmad](https://github.com/w-ahmad) in [#165](https://github.com/w-ahmad/WinUI.TableView/pull/165)
- Fixed auto row height issue by [@w-ahmad](https://github.com/w-ahmad) in [#165](https://github.com/w-ahmad/WinUI.TableView/pull/165)
- Fixed auto header row height issue by [@w-ahmad](https://github.com/w-ahmad) in [#184](https://github.com/w-ahmad/WinUI.TableView/pull/184)
- Fixed `WebView2Loader.dll` not found issue by [@w-ahmad](https://github.com/w-ahmad) in [#182](https://github.com/w-ahmad/WinUI.TableView/pull/182)
- Improved auto generate column process by [@w-ahmad](https://github.com/w-ahmad) in [#163](https://github.com/w-ahmad/WinUI.TableView/pull/163)
- Fixed `DataContext` of `ContextFlyouts` on Uno by [@w-ahmad](https://github.com/w-ahmad) in [#186](https://github.com/w-ahmad/WinUI.TableView/pull/186)

## Breaking Changes

- `TableView.RowHeight` default value changed to `double.NaN` (was `40d`)
- `TableView.HeaderRowHeight` default value changed to `double.NaN` (was `32d`)
