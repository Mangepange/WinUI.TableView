# v1.2.0-preview2

> **Preview Release** · July 9, 2024 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.2.0-preview2) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.2.0-preview2)

## What's New

- Introduced `CellTemplateSelector` and `EditingTemplateSelector` properties in `TableViewTemplateColumn`
- There won't be a separator at the end of the options flyout when export options are hidden

## Bug Fixes

- Fixed an issue where `ComboBox` of `TableViewComboBoxColumn` was not able to open
- Fixed an issue with keyboard shortcuts not working at startup [#25](https://github.com/w-ahmad/WinUI.TableView/issues/25)
- Fixed cell selection issue when there is an active column filter
- Fixed cell layout issue when the size was narrower than the desired width of a cell
- Fixed an issue where cells were not able to select when there is only one column

## Known Issues

- Duplicate items in `ItemsSource` can lead to unexpected cell or row navigation behavior
