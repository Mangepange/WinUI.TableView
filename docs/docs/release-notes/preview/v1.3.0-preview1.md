# v1.3.0-preview1

> **Preview Release** · February 8, 2025 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.3.0-preview1) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.3.0-preview1)

## Upgraded to .NET 8

## What's New

- **Gridlines**: Added support for customizable horizontal and vertical gridlines. These gridlines are enabled by default but can be turned off using `HeaderGridLinesVisibility` and `GridLinesVisibility` properties. Related properties: `VerticalGridLinesStrokeThickness`, `HorizontalGridLinesStrokeThickness`, `VerticalGridLinesStroke`, `HorizontalGridLinesStroke`. [#74](https://github.com/w-ahmad/WinUI.TableView/pull/74)
- **Alternate Row Colors**: Added support for alternate row background and foreground colors via `AlternateRowBackground` and `AlternateRowForeground` properties. [#83](https://github.com/w-ahmad/WinUI.TableView/pull/83), [#21](https://github.com/w-ahmad/WinUI.TableView/pull/21)
- **Cell & Row Context Flyout**: Added support for context flyouts on cells and rows via `CellContextFlyout` and `RowContextFlyout`. [#85](https://github.com/w-ahmad/WinUI.TableView/pull/85), [#38](https://github.com/w-ahmad/WinUI.TableView/pull/38)
- **Custom Sorting and Filtering**: Implemented `IColumnFilterHandler` and introduced `Sorting` and `ClearSorting` events. [#98](https://github.com/w-ahmad/WinUI.TableView/pull/98), [#30](https://github.com/w-ahmad/WinUI.TableView/pull/30), [#55](https://github.com/w-ahmad/WinUI.TableView/pull/55), [#77](https://github.com/w-ahmad/WinUI.TableView/pull/77), [#92](https://github.com/w-ahmad/WinUI.TableView/pull/92)
- **Three-State Column Sorting**: Enhanced sorting with Ascending, Descending, and None states. [#98](https://github.com/w-ahmad/WinUI.TableView/pull/98), [#37](https://github.com/w-ahmad/WinUI.TableView/pull/37)
- **Corner Button Mode**: Implemented `CornerButtonMode` to choose corner button functionality. `ShowOptionsButton` property is replaced with `CornerButtonMode`. [#80](https://github.com/w-ahmad/WinUI.TableView/pull/80), [#23](https://github.com/w-ahmad/WinUI.TableView/pull/23)
- **Tag Property**: `Tag` property added to `TableViewColumn`. [#84](https://github.com/w-ahmad/WinUI.TableView/pull/84), [#49](https://github.com/w-ahmad/WinUI.TableView/pull/49)
- **Customize Column Headers**: Column headers style via `TableView.ColumnHeaderStyle` and `TableViewColumn.HeaderStyle`
- **Customize Cell Styles**: Cell style via `TableView.CellStyle` and `TableViewColumn.CellStyle`
- **Element & Editing Element Style**: Set through `ElementStyle` and `EditingElementStyle` on `TableViewBoundColumn`

## New Column Types

- **TableViewDateColumn** – Supports `DateTimeOffset`, `DateTime`, and `DateOnly`. [#87](https://github.com/w-ahmad/WinUI.TableView/pull/87)
- **TableViewTimeColumn** – Supports `DateTimeOffset`, `DateTime`, `TimeOnly`, and `TimeSpan`. [#87](https://github.com/w-ahmad/WinUI.TableView/pull/87)

## What's Changed

- `AdvancedCollectionView` from Community Toolkit replaced with custom `CollectionView`. [#91](https://github.com/w-ahmad/WinUI.TableView/pull/91)

## Fixes

- Fixed select all check box fast click issue [#61](https://github.com/w-ahmad/WinUI.TableView/pull/61), [#81](https://github.com/w-ahmad/WinUI.TableView/pull/81)
- Crashes caused by empty property paths in column bindings are fixed [#42](https://github.com/w-ahmad/WinUI.TableView/pull/42), [#91](https://github.com/w-ahmad/WinUI.TableView/pull/91)
- Fixed a crash when there are null values in primitive type columns [#89](https://github.com/w-ahmad/WinUI.TableView/pull/89), [#91](https://github.com/w-ahmad/WinUI.TableView/pull/91)

## Breaking Changes

- `ShowOptionsButton` property has been removed
- `CollectionView` property is now of type `ICollectionView`

## Known Issues

- Changing grid line thickness or visibility at runtime may lead to unexpected layouts. Due to [microsoft/microsoft-ui-xaml#9860](https://github.com/microsoft/microsoft-ui-xaml/issues/9860)
