# v1.0.0-preview4

> **Preview Release** · March 26, 2024 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.0.0-preview4) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.0.0-preview4)

## What's New

- `AutoGenerateColumns` is now available (enabled by default). To turn it off, set `AutoGenerateColumns = false`
- Export options are turned off by default. To turn them on, set `ShowExportOptions = true`
- Exposed methods to implement custom export: `GetAllRowsContent(bool includeHeaders, char separator)` and `GetSelectedRowsContent(bool includeHeaders, char separator)`
- Improved detection of open flyouts on cell focus lost
- Fixed issues where keyboard accelerators were not working properly
