## What's New

- Added Slovak translations by [@marianl79](https://github.com/marianl79) in [#173](https://github.com/w-ahmad/WinUI.TableView/pull/173)
- Added .NET 10 target by [@w-ahmad](https://github.com/w-ahmad) in [#257](https://github.com/w-ahmad/WinUI.TableView/pull/257)
- Removed `CommunityToolkit.WinUI.Behaviors` dependency by [@w-ahmad](https://github.com/w-ahmad) in [#258](https://github.com/w-ahmad/WinUI.TableView/pull/258)
- Added items count in filter flyout by [@w-ahmad](https://github.com/w-ahmad) in [#262](https://github.com/w-ahmad/WinUI.TableView/pull/262)
- A major performance boost for sort and filter mechanism by [@w-ahmad](https://github.com/w-ahmad) in [#277](https://github.com/w-ahmad/WinUI.TableView/pull/277)
- Added `SortMemberPath` to define sort property by [@cricketthomas](https://github.com/cricketthomas) in [#279](https://github.com/w-ahmad/WinUI.TableView/pull/279)

- Prevent `MenuFlyout` click action on Space key press by [@w-ahmad](https://github.com/w-ahmad) in [#259](https://github.com/w-ahmad/WinUI.TableView/pull/259)

- Fixed unexpected cell edit states when reloading in a frame by [@w-ahmad](https://github.com/w-ahmad) in [#273](https://github.com/w-ahmad/WinUI.TableView/pull/273)

- Fixed horizontal scroll issue where sometimes the header row gets frozen by [@w-ahmad](https://github.com/w-ahmad) in [#299](https://github.com/w-ahmad/WinUI.TableView/pull/299)
