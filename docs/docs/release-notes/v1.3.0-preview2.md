# v1.3.0-preview2

> **Preview Release** · April 3, 2025 · [NuGet](https://www.nuget.org/packages/WinUI.TableView/1.3.0-preview2) · [GitHub Release](https://github.com/w-ahmad/WinUI.TableView/releases/tag/v1.3.0-preview2)

## Uno Platform Support

This release adds Uno Platform support to WinUI.TableView, enabling compatibility with WASM, macOS, and Linux.

## What's New

- Added `RowMinHeight` property [#110](https://github.com/w-ahmad/WinUI.TableView/pull/110)
- Added localization support; thanks to [@VisualAlf](https://github.com/VisualAlf) for de-DE translations and [@arthurits](https://github.com/arthurits) for es-ES translations. [#113](https://github.com/w-ahmad/WinUI.TableView/pull/113), [#116](https://github.com/w-ahmad/WinUI.TableView/pull/116), [#126](https://github.com/w-ahmad/WinUI.TableView/pull/126)
- Added `PlaceholderText` property for `TimeColumn`
- Exposed `ScrollRowIntoView` and `ScrollCellIntoView` methods [#123](https://github.com/w-ahmad/WinUI.TableView/pull/123)

## What's Changed

- Fixed vertical grid lines visibility issue
- Fixed alternate row colors issue with theme changes [#95](https://github.com/w-ahmad/WinUI.TableView/issues/95)
- Fixed date column placeholder text issue
- Fire `ContextFlyoutOpened` even if `Flyout` is null by [@VisualAlf](https://github.com/VisualAlf) in [#108](https://github.com/w-ahmad/WinUI.TableView/pull/108)
- Changed default types for date and time column in [#124](https://github.com/w-ahmad/WinUI.TableView/pull/124)
